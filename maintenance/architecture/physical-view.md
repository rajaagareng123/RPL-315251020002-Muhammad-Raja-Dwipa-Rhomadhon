# Physical View

## Penjelasan

Physical View menunjukkan infrastruktur fisik yang digunakan oleh Aplikasi Pemesanan Tiket. Pengguna mengakses sistem melalui browser atau aplikasi mobile yang terhubung ke internet. Permintaan dari pengguna diterima oleh Web Server, kemudian diteruskan ke Application Server untuk diproses sesuai fungsi yang dijalankan, seperti login, pemesanan tiket, dan pembayaran. Seluruh data sistem, seperti data pengguna, tiket, jadwal, dan transaksi, disimpan di Database MySQL. Semua komponen saling terhubung melalui jaringan internet sehingga sistem dapat diakses dengan mudah dan bekerja secara terintegrasi.
